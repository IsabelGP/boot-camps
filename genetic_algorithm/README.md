## Setup

### Install pyGene

1. Clone pyGene repo https://github.com/blaa/PyGene
1. In the home directory run `python setup.py install`

### Generate documentation

1. Download Eypdoc from http://epydoc.sourceforge.net/
1. Unzip and then
```
sudo make install
sudo make doc
```

### View documentation
1. `cd PyGene/doc`
1. `open index.html`
